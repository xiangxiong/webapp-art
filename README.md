## 前言
此项目涉及微信授权登录、商品分类、社区、商家入驻、购物车、下单、支付、商品详情总计有55个单页的H5 微商城。整个项目还在不断的完善和不断优化当中.

## 技术栈
react + redux + react-router + es6 + axios + sass + antd-mobile

## 项目运行
```
git clone https://github.com/xiangxiong/webapp-art.git

cd webapp-art

yarn install

yarn start
```
