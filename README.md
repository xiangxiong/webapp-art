
> 技术架构:
 * 1、react + redux + webpack 4.0. 
 * 2、组件库 antd-mobile.
 * 3、样式 sass.
 * 4、react ssr.

> 目录结构
* pages  页面.
  * home 首页
  * user 用户中心.
  * shop 艺商城.
  * master 艺术大家.
  * cart 购物车.
  * user-login 用户登录.
  * community 社区.
  
* components   公用组件库.

* utils  公用类库.

* statics  字体库和图片资料.

> 约束
* css 写法符合 bem 规范.
